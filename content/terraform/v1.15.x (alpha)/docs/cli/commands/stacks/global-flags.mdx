# terraform stacks CLI global flags reference

You can use the following flags with all commands in the Terraform Stacks CLI.

- `-chdir`: Overwrite the current directory as the target for the root configuration.
    - Data type: string
    - Default: Current working directory

- `-plugin-cache-dir`: Override the default directory where the stack plugin binary is cached.
    - Data type: string
    - Default: Default plugin cache directory

- `-no-color`: Disable color output.
    - Data type: boolean
    - Default: false

- `-usage`: Show usage overview or the usage for a specified subcommand if provided.

- `-stacks-version`: An alias for the "version" subcommand.
