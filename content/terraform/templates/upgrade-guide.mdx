# Atualizando para o Terraform v__VERSION__

-> **Dica:** Use o seletor de versão para visualizar os guias de atualização
para versões mais antigas do Terraform.

O Terraform v__VERSION__ é uma versão secundária da série estável Terraform
v1.0.

O Terraform v__VERSION__ respeita as
[Promessas de Compatibilidade do Terraform v1.0](https://developer.hashicorp.com/terraform/language/v1-compatibility-promises).

Consulte
[o changelog completo](https://github.com/hashicorp/terraform/blob/v__VERSION__/CHANGELOG.md)
para obter mais detalhes.
Se você encontrar algum problema durante a atualização que não esteja abordado
neste guia, inicie um novo tópico no
[fórum da comunidade Terraform](https://discuss.hashicorp.com/c/terraform-core)
para discuti-lo.
